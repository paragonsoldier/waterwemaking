This guide is written as reference for Part 1 of the book [Automate the Boring Stuff with Python, 2nd Edition](https://nostarch.com/automatestuff2) which is published by [No Starch Press](https://nostarch.com/). Part 1 of this book covers Python Programming Basics which includes the following six chapters:
- Chapter 1: Python Basics
- Chapter 2: Flow Control
- Chapter 3: Functions
- Chapter 4: Lists
- Chapter 5: Dictionaries and Structuring Data
- Chapter 6: Manipulating Strings

Part 2 of the book isn't covered in this guide, because it focuses on project based work. Purchased the book to read the additional chapters covering information on Regular Expressions, Input Validation, File Manipulation, Web Scraping, Spreadsheets, and Image Manipulation.
